# 🎉 Python Quiz Game

A **fun and interactive command-line quiz game** built with Python!  
Test your knowledge, answer questions, and see how high you can score! 🏆

---

## ✨ Features

- 🖥️ Command-line interface – simple and lightweight  
- ❓ Multiple-choice quiz questions  
- ✅ Instant feedback on answers  
- 🏅 Automatic score calculation  
- 📂 Single Python file – easy to run

---

## 🛠️ Built With

- 🐍 Python

---

## 📝 How It Works

1. 🎮 Start the quiz  
2. 🤔 Answer each question  
3. ✅ The game checks if your answer is correct  
4. 🏁 See your final score at the end

---

## 🚀 How to Run

1. Make sure **Python** is installed on your computer  
2. Clone this repository:

```bash
git clone https://github.com/yourusername/Python_Quiz_Game.git
```

3. Navigate into the project folder:

```bash
cd Python_Quiz_Game
```

4. Run the quiz:

```bash
python quiz_game.py
```
